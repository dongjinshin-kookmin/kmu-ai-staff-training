# 화이트 스페이스 (Whitespace First) — 대학 보고서 발간용 디자인 가이드

## 1. 가이드 개요

**가이드 이름** — 화이트 스페이스 (Whitespace First)

**디자인 철학** — 덜어낸다. 여백은 남은 공간이 아니라 설계된 공간이다. 지면 위에서 가장 강한 요소는 잉크가 아니라 침묵이다. 콘텐츠가 주인공이 되도록 디자인은 뒤로 물러선다(deference). 한 페이지에는 하나의 메시지만 둔다. 장식을 더해 설득하지 않고, 비워서 신뢰하게 한다.

**참고한 디자인 시스템** — Apple Human Interface Guidelines의 미학적 원칙(Deference, Clarity, Depth)을 인쇄물로 번안했다. 화면의 여백·계층·절제 개념을 A4 지면의 마진·타입 스케일·모노톤 팔레트로 옮긴다. 스위스 인터내셔널 타이포그래피의 그리드 규율을 뼈대로 삼되, 격자를 드러내지 않고 여백으로 감춘다.

**어울리는 보고서** — 대외 홍보용, 총장·이사회 보고용 프리미엄 보고서. 사진 품질이 좋은 행사·행사 결과보고서, 준공·기념식·국제 컨퍼런스 화보형 보고서에 최적이다. 국민대학교 각 부서가 대외로 내보내는 결과보고서에서 "격(格)"을 만들어야 할 때 쓴다. 데이터가 빽빽한 실무 정산 보고서에는 적합하지 않다 — 그런 문서는 정보 밀도형 가이드를 쓸 것.

## 2. 디자인 원칙

**원칙 1 — 여백이 먼저다 (Whitespace First).** 레이아웃은 콘텐츠를 배치하는 일이 아니라 여백을 배치하는 일이다. 요소를 놓기 전에 빈 공간부터 확보한다. 페이지 사방 여백은 최소 30mm, 프리미엄 표지·간지는 40mm 이상. 여백이 부족하면 요소를 옮기지 말고 요소를 덜어낸다.

**원칙 2 — 한 페이지, 한 메시지.** 한 페이지는 하나의 생각만 전달한다. 두 개의 주장이 있으면 두 페이지로 나눈다. 지면을 아끼려 밀어 넣지 않는다. 페이지 수가 늘어도 좋다 — 여백은 비용이 아니라 품질이다.

**원칙 3 — 크기로 위계를 만든다.** 색이나 굵기가 아니라 크기 대비로 계층을 세운다. 제목 40pt와 본문 10pt의 극단적 대비가 위계를 만든다. 굵기는 최소한으로 쓴다. 가늘고 큰 글자가 크고 굵은 글자보다 우아하다.

**원칙 4 — 컬러는 정보가 아니라 강조다.** 지면은 거의 무채색으로 유지한다. 색은 데이터를 구분하려고 쓰지 않는다. 한 페이지에서 진정으로 중요한 단 하나를 가리킬 때만 포인트 컬러를 허용한다. 색이 두 곳에 나타나면 하나는 잘못 쓴 것이다.

**원칙 5 — 디자인은 물러선다 (Deference).** 테두리·그림자·그라데이션·장식 괘선을 쓰지 않는다. 구분은 선이 아니라 간격으로 한다. 표에 격자를 그리지 않고, 콜아웃에 배경색을 깔지 않는다. 사용자가 디자인을 알아채지 못할 때 디자인은 성공한다.

## 3. 컬러 시스템

무채색 5단계 그레이 스케일에 포인트 컬러 하나만 더한다. 지면의 95% 이상은 화이트와 그레이가 차지한다.

| 역할 | 이름 | HEX | 용도 |
|---|---|---|---|
| Primary (교체 슬롯) | Point | `#0071E3` | 강조 단 한 곳. 대학 CI 주색으로 교체 |
| Secondary | Ink | `#1D1D1F` | 제목·핵심 텍스트 (순흑 아님) |
| Neutral 1 | Graphite | `#4B4B4F` | 본문 텍스트 |
| Neutral 2 | Slate | `#86868B` | 캡션·보조 텍스트 |
| Neutral 3 | Mist | `#D2D2D7` | 얇은 구분선·표 헤더 밑줄 |
| Neutral 4 | Paper | `#F5F5F7` | 간지·박스 배경(아주 드물게) |
| Base | White | `#FFFFFF` | 지면 바탕 |
| Semantic 긍정 | Positive | `#2E7D32` | 차트에서만, 필요 시 |
| Semantic 주의 | Caution | `#B26A00` | 차트에서만, 필요 시 |
| Semantic 부정 | Negative | `#C62828` | 차트에서만, 필요 시 |

**Primary 교체 슬롯** — `#0071E3`는 임시 포인트 컬러다. 발간 시 국민대학교 공식 CI 주색상 HEX로 교체한다. 정확한 CI 값은 대학 홍보·브랜드 담당 부서의 CI 규정집에서 확인해 넣을 것. **본 가이드는 국민대학교 공식 CI 색상 값을 단정하지 않는다.** CI 색이 채도·명도가 높아 대비가 부족하면 텍스트가 아닌 그래픽 요소(선·도형)에만 쓴다.

**사용 비율 규칙** — White/Paper 배경 **90% 이상**, Ink/Graphite/Slate 텍스트 **8% 내외**, Mist 구분선 **1% 내외**, Point 컬러 **1% 이하**. 포인트 컬러가 한 페이지에서 차지하는 면적은 명함 크기(약 90×55mm)를 넘지 않는다.

**명도 대비(WCAG 2.1)** — 본문 텍스트는 AA 기준 4.5:1 이상을 지킨다. Graphite `#4B4B4F` / White = 8.9:1, Ink `#1D1D1F` / White = 16.1:1 로 모두 AAA(7:1) 충족. Slate `#86868B` / White = 3.9:1 은 4.5:1 미만이므로 **캡션·라벨 등 큰 글자(18pt 이상 또는 14pt Bold)에만** 사용하고 본문에는 쓰지 않는다. Point `#0071E3` / White = 3.5:1 이므로 포인트 컬러 위에 흰 글자를 얹지 말고, 색은 도형·아이콘·굵은 큰 글자에만 쓴다.

**인쇄용 CMYK 근사값** — 화면 HEX를 인쇄로 옮길 때 아래 근사값을 기준으로 잡되, 최종 색교(color proof)로 확정한다. 무채색은 K 단일 채널로만 조판해 4색 겹침에 의한 회색 얼룩을 막는다.

| 이름 | HEX | CMYK 근사 |
|---|---|---|
| Ink | `#1D1D1F` | 0 / 0 / 0 / 92 |
| Graphite | `#4B4B4F` | 0 / 0 / 0 / 78 |
| Slate | `#86868B` | 0 / 0 / 0 / 52 |
| Mist | `#D2D2D7` | 0 / 0 / 0 / 16 |
| Paper | `#F5F5F7` | 0 / 0 / 0 / 3 |
| Point(임시) | `#0071E3` | 88 / 55 / 0 / 0 |

## 4. 타이포그래피

**서체** — 한글·영문 모두 **Pretendard**(무료, SIL Open Font License) 단일 패밀리로 통일한다. 대학 실무에서 설치·배포가 쉽고 굵기 단계(Thin~Black)가 풍부해 크기·굵기 대비 전략에 맞는다. 숫자·표에는 Pretendard의 등폭(tabular) 숫자 기능을 켠다. 서체를 두 개 섞지 않는다 — 한 패밀리의 굵기 대비만으로 위계를 만든다.

**타입 스케일**

| 용도 | 서체·굵기 | 크기(pt) | 행간 | 자간 |
|---|---|---|---|---|
| 표지 제목 | Pretendard Light | 44 | 118% | -2.0% |
| 표지 부제 | Pretendard Regular | 15 | 150% | 0% |
| 대제목(H1) | Pretendard Light | 30 | 130% | -1.5% |
| 중제목(H2) | Pretendard Regular | 19 | 140% | -0.5% |
| 소제목(H3) | Pretendard Medium | 13 | 150% | 0% |
| 본문 | Pretendard Regular | 10 | 175% | 0% |
| 강조 본문 | Pretendard Medium | 10 | 175% | 0% |
| 캡션·주석 | Pretendard Regular | 8 | 150% | +0.5% |
| 표 내부 | Pretendard Regular | 9 | 140% | 0% |
| 페이지 번호 | Pretendard Regular | 8 | — | +2.0% |

**위계 규칙** — 크기 대비를 극단적으로 둔다. 표지 제목 44pt와 본문 10pt는 4.4배 차이다. 굵기는 위계의 보조 수단일 뿐, 큰 제목일수록 오히려 **가늘게(Light)** 쓴다. 본문 행간 175%는 이 가이드의 서명(署名)이다 — 넉넉한 행간이 여백 철학을 문단 안에서 재현한다. 한 페이지에 3단계를 넘는 위계를 두지 않는다. 밑줄·이탤릭·전각 강조를 쓰지 않는다.

**문단 조판 규칙** — 첫 줄 들여쓰기를 쓰지 않는다. 문단 구분은 들여쓰기가 아니라 문단 사이 여백(본문 크기의 1배, 약 4.4mm)으로 한다. 본문은 왼끝 맞춤·오른쪽 흘림(ragged right)으로 조판하고, 양끝 맞춤(justify)은 쓰지 않는다 — 강제 양끝 맞춤이 만드는 단어 간 흰 강(river)이 여백의 순도를 흐린다. 한 문단은 최대 5줄을 넘기지 않도록 끊는다. 하이픈 자동 분철을 끈다.

## 5. 레이아웃·그리드

**판형** — A4 세로(210 × 297mm) 기준.

**여백(마진)** — 본문 페이지: 상 35mm / 하 30mm / 좌·우 각 30mm. 표지·간지: 사방 40mm 이상. 하단 여백 안 12mm 지점에 페이지 번호를 둔다. 여백은 이 가이드에서 가장 중요한 규격이며, 콘텐츠가 많아도 절대 침범하지 않는다.

**단(column) 구성** — 기본 **1단**. 텍스트 열 폭은 최대 150mm로 제한해, 여백이 30mm보다 넓어지더라도 한 줄에 한글 약 40자, 영문 약 65자 이상 넘지 않게 한다(가독 최적 행장 유지). 데이터·병렬 항목이 많은 페이지에 한해 **2단**(단 폭 각 70mm, 단 간격 10mm)을 허용한다. 3단 이상은 이 가이드에서 쓰지 않는다.

**베이스라인 그리드** — 본문 행간 기준 **17.5pt(약 6.17mm) 베이스라인 그리드**에 모든 텍스트 기준선을 맞춘다. 제목·이미지·표는 이 그리드의 배수 위치에 정렬한다. 수직 리듬이 여백의 규율을 완성한다.

**안전 여백** — 재단선(트림) 안쪽 5mm는 텍스트·핵심 요소 금지 구역. 풀블리드 사진은 재단선 밖 3mm 도련(bleed)까지 채운다.

**여백 규격 요약**

| 위치 | 본문 페이지 | 표지·간지 | 풀블리드 사진 |
|---|---|---|---|
| 상 | 35mm | 40mm 이상 | 0(도련 3mm) |
| 하 | 30mm | 40mm 이상 | 0(도련 3mm) |
| 좌·우 | 각 30mm | 각 40mm 이상 | 0(도련 3mm) |
| 텍스트 열 폭 | ≤150mm | — | 사진 위 안전영역 내 |
| 도련(bleed) | — | — | 사방 3mm |

**풀블리드 사진 페이지 규칙** — 이 가이드의 핵심 장치다. 화보급 사진 한 장을 지면 전체에 재단선 밖 3mm 도련까지 채운다. 규칙은 다음과 같다. (1) 한 페이지에 사진은 **한 장만** — 여러 장을 풀블리드로 겹치지 않는다. (2) 사진 위 텍스트는 캡션 한 줄 또는 제목 한 줄까지만 허용하고, 재단선 안쪽 25mm 안전영역 안에 배치한다. (3) 텍스트를 얹는 자리는 사진에서 디테일이 적고 어두운 영역을 고르고, 하단 배치 시 하단 30% 높이에 Ink→투명 세로 그라데이션(불투명도 60%→0%)을 깔아 가독성을 확보한다. (4) 사진 해상도는 실제 출력 크기 기준 **300dpi 이상**(A4 전면이면 최소 2480×3508px). (5) 마주 보는 양면(스프레드) 풀블리드는 접지(gutter) 부분에 인물 얼굴·핵심 피사체가 걸리지 않게 한다. (6) 풀블리드 사진 페이지 다음 장은 여백이 넉넉한 순백 페이지로 받아 호흡을 만든다.

## 6. 컴포넌트 스타일

**표** — 세로 괘선을 전부 없앤다. 헤더 아래 0.5pt Mist(`#D2D2D7`) 가로선 하나, 표 맨 아래 0.5pt Ink 가로선 하나만 남긴다. 행 사이 구분은 선이 아니라 **행 높이(최소 8mm)와 셀 상하 여백(각 2.5mm)**으로 만든다. 셀 배경색·얼룩무늬(zebra)를 쓰지 않는다. 숫자는 우측 정렬·등폭 숫자, 텍스트는 좌측 정렬.

**차트** — 격자선·테두리·범례 상자를 지운다. 축은 X축 기저선 하나만 0.5pt Mist로 남기고 Y축 선은 없앤다. 값 라벨을 데이터 요소 옆에 직접 붙여 범례를 대체한다. 막대·선은 기본 Graphite/Slate로 그리고, 강조할 단 하나의 계열에만 Point 컬러를 쓴다. 3D·그림자·그라데이션 금지.

**인용·콜아웃 박스** — 상자를 그리지 않는다. 배경색·테두리 없이, 좌측에 2pt Point 컬러 세로선 하나만 두고 본문보다 2pt 크게(12pt), Slate가 아닌 Graphite로 조판한다. 위아래 여백을 본문의 2배(약 12mm)로 벌려 여백으로 구분한다.

**아이콘** — 선(line) 스타일, 1.5pt 굵기, 단색(Ink 또는 Point) 픽토그램만. 채움(filled)·다색 아이콘 금지. 크기는 20 × 20mm 그리드에 맞추고, 아이콘 주위 최소 8mm 여백을 확보한다. 장식용 아이콘은 쓰지 않고, 의미를 돕는 곳에만 절제해 쓴다.

**선(rule) 규격** — 이 가이드는 선을 극도로 아낀다. 허용하는 선은 아래 넷뿐이며, 그 외 장식 괘선·박스 테두리·구분 박스는 금지한다.

| 쓰임 | 굵기 | 색 |
|---|---|---|
| 표 헤더 밑줄 | 0.5pt | Mist `#D2D2D7` |
| 표 하단·합계 구분선 | 0.5pt | Ink `#1D1D1F` |
| 콜아웃 좌측 세로선 | 2pt | Point(교체) |
| 표지 제목 아래 강조선 | 3pt / 폭 40mm | Point(교체) |

**사진 처리** — 원본 색을 살리되 과한 필터·테두리·둥근 모서리·그림자를 넣지 않는다. 사진은 직각 모서리로 배치한다. 페이지 폭에 걸치는 풀블리드(7절 규칙) 또는 그리드 정렬 중 하나를 택한다. 그리드 정렬 시 사진 사이 간격(gutter)은 5mm로 일정하게 두고, 사진 종횡비를 한 페이지 안에서 통일한다(예: 3:2 또는 4:3 중 하나). 사진 위에 텍스트를 얹을 때는 이미지 위 어두운 영역을 골라 배치하고, 필요 시 하단 30% 높이에 Ink→투명 그라데이션(불투명도 60%→0%)을 깔아 가독성을 확보한다. 캡션은 사진 바로 아래 2mm 간격, 8pt Slate로 좌측 정렬.

## 7. 표지·목차·간지 디자인 지침

**표지** — 사방 40mm 여백. 상단 35% 영역은 완전히 비운다. 제목은 세로 중앙보다 약간 위(상단에서 45% 지점)에 좌측 정렬로 놓고 Pretendard Light 44pt. **텍스트는 최대 3줄** — 보고서 제목 1줄, 부제 1줄, 발간 주체·일자 1줄. 그 이상 넣지 않는다. 로고는 상단 좌측 여백 안 또는 하단 우측에 작게(높이 10mm 이내) 배치. 배경은 순백(`#FFFFFF`)이거나, 사진 표지라면 화보급 풀블리드 사진 한 장 위에 텍스트 3줄. 포인트 컬러는 제목 아래 폭 40mm·굵기 3pt의 짧은 선 하나로만 등장할 수 있다.

**목차** — 1단, 사방 여백 35mm 이상. 항목 사이 간격을 본문의 2배(최소 14mm)로 벌린다. 점선 리더(leader)를 쓰지 않고, 항목명은 좌측·페이지 번호는 우측에 두되 그 사이를 넉넉한 여백으로 비운다. 장 번호는 Point 컬러 또는 Slate로 작게. 목차는 한 페이지를 넘기지 않도록 항목을 대분류 중심으로 추린다.

**간지(섹션 표지)** — 장이 바뀔 때 삽입. 페이지 전체를 비우고 화면 중앙에서 상단 40% 지점에 장 번호(Pretendard Light 30pt, Point 컬러)와 장 제목(Pretendard Light 30pt, Ink) 두 줄만 놓는다. 나머지는 전부 여백. 배경 Paper(`#F5F5F7`)를 아주 옅게 깔 수 있으나 기본은 순백. 간지에는 본문·설명을 넣지 않는다 — 숨을 쉬는 페이지다.

## 8. 적용 예시 — '워크숍 결과보고서'에 적용하기

국민대학교 부서가 주최한 워크숍의 결과보고서를 이 가이드로 편집하는 경우, 페이지 구성을 다음과 같이 잡는다.

**1) 표지** — 워크숍 현장 사진 중 가장 좋은 한 장을 풀블리드로 깔고, 하단 30%에 Ink 그라데이션. 텍스트 3줄: "2026 ○○워크숍 결과보고서" / "주제 한 줄" / "국민대학교 ○○부서 · 2026.07". 로고 하단 우측 높이 10mm. 제목은 Pretendard Light 44pt 흰색.

**2) 개요 페이지** — 사방 여백 35mm, 한 페이지 한 메시지. 상단에 H1 30pt "워크숍 개요", 그 아래 넉넉한 여백, 본문 10pt·행간 175%로 목적·일시·장소·참가 인원을 5~6줄로 압축. 핵심 수치(참가자 수 등) 하나만 Point 컬러 큰 숫자(30pt)로 강조. 표지 맥락에서 국민대학교 부서 주최임을 자연스럽게 명시.

**3) 일정표 페이지** — 6절의 표 규칙 적용. 세로 괘선 없이 헤더 아래·표 하단 가로선만. 시간 열은 좌측, 세션명 우측. 행 높이 8mm, 셀 상하 여백 2.5mm. 시간은 등폭 숫자 우측 정렬. 오전/오후 구분은 선이 아니라 그룹 사이 여백 12mm로.

**4) 만족도 차트 페이지** — 한 페이지 하나의 차트. 만족도 항목별 가로 막대. 격자·테두리·범례 상자 제거, 값 라벨을 막대 끝에 직접 표기. 전체 종합만족도 막대 하나만 Point 컬러, 나머지 항목은 Graphite. 차트 위 H2 19pt로 "참가자 만족도", 아래 캡션 8pt로 "n=응답자 수, 5점 척도".

**5) 예산표 페이지** — 6절 표 규칙. 항목·집행액·비율 3열, 세로 괘선 없음. 금액은 등폭 숫자 우측 정렬, 천 단위 콤마. 합계 행 위에 0.5pt Ink 가로선 하나로 구분하고 합계는 Medium 굵기. 셀 배경색 없이 여백으로 위계를 만든다. 비고가 길면 표 밖 캡션 8pt로 뺀다.

전체적으로 각 꼭지를 한 페이지씩 배분하고, 장이 바뀌는 곳(개요→결과→정산)마다 간지를 넣어 호흡을 준다. 페이지 수가 늘어도 밀어 넣지 않는다.

## 9. 발간 전 점검 체크리스트

1. 모든 본문 페이지의 사방 여백이 30mm 이상인가 (표지·간지 40mm 이상).
2. 한 페이지에 메시지가 하나인가 — 두 주장이 섞인 페이지는 없는가.
3. Primary 슬롯의 임시 색 `#0071E3`을 국민대학교 공식 CI 주색으로 교체했는가.
4. 한 페이지에서 포인트 컬러가 단 한 곳(면적 명함 크기 이하)에만 쓰였는가.
5. 본문 텍스트 대비가 WCAG AA(4.5:1) 이상인가 — Slate를 본문에 쓰지 않았는가.
6. 서체가 Pretendard 한 패밀리로 통일되고, 큰 제목은 Light로 조판됐는가.
7. 본문 행간이 175%이고 텍스트 기준선이 17.5pt 베이스라인 그리드에 맞는가.
8. 표에 세로 괘선·셀 배경색·얼룩무늬가 없는가 (가로선 최소한만).
9. 표지 텍스트가 3줄 이하이고, 풀블리드 사진의 도련(bleed) 3mm가 확보됐는가.
10. 그림자·테두리·둥근 모서리·그라데이션 장식이 지면에서 제거됐는가.
